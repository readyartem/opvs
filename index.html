Frequently asked questions

How secure is my company data? +

Enterprise-grade by default. OpenClaw supports deep VPC integrations, SOC2 compliance, and zero-data-retention modes. Your proprietary data never trains our base models.

Do I need coding experience? +

None at all. OPVS Agents are configured using natural language SOUL.md files and instructed just like human employees via the AgentBoard chat interface.

How is OPVS different from OpenAI's GPTs? +

GPTs are reactive chat wrappers. OPVS Agents are proactive, asynchronous workers running on the OpenClaw VM. They can execute background tasks, manage cloud infrastructure, and retain long-term memory across your entire workspace.

What happens if an agent makes a mistake? +

Agents feature built-in 'Confidence Thresholds'. If an agent is unsure about a destructive action (like deleting a database table or sending a mass email), it pauses execution and pings a human manager for approval.
